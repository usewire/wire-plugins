# Wire Plugins

Official [Claude Code](https://docs.anthropic.com/en/docs/claude-code) plugin marketplace from [Wire](https://usewire.io).

## Install

```bash
claude plugin marketplace add usewire/wire-plugins
```

## Available Plugins

### wire-memory

Persistent memory for AI agents. Connect to a Wire container to remember decisions, patterns, and context across sessions. Zero dependencies.

```bash
claude plugin install wire-memory@wire-plugins
```

Then run `/wire-memory:connect` inside Claude Code to authenticate and link a container.

[Source](https://github.com/usewire/wire-memory) | [Wire](https://usewire.io)
